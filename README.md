# TooltipMod
Mod for MIMESIS that gives the hotbar a tooltip listing the item name, value and weight
## REQUIREMENTS:
MelonLoader https://github.com/LavaGang/MelonLoader/

MimicAPI https://github.com/NeoMimicry/MimicAPI

## Build Instructions:
Install MelonLoader and MimicAPI to your install of MIMESIS
Change path in Directory.Build.props to your install of MIMESIS
Build
